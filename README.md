# High Rollers

## Project

### Description

Sixth challenge project of [100 Days of SwiftUI](https://www.hackingwithswift.com/100/swiftui)

This application lets users roll dice, and also let them see results from previous rolls.

## Development guidelines

### Scope

* Xcode: 13.4.1 (13F100)
* min iOS: 15.5+
* Swift: 5.0
